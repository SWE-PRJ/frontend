## 실행방법

### IDE에서 실행
1. 먼저 백엔드 어플리게이션을 실행한다.
2. git clone

```bash
$ git clone https://github.com/SWE-PRJ/frontend.git
```
3. build

```bash
$ npm run build
```
4. run
```bash
$ npm run start
```
5. 다음 주소로 접속한다.
http://localhost:3000/
